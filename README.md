# @momoko/eslint-config

## 安装

```bash
# for js
yarn add -D @momoko/eslint-config-js
# for vue
yarn add -D @momoko/eslint-config-vue
# for react
yarn add -D @momoko/eslint-config-react
```

## 使用

> package.json 中添加

```js
  "eslintConfig": {
    "extends": [
      "@momoko/js",
      "@momoko/vue",
      "@momoko/react"
    ]
  },
```
